## ブログ記事の作成手順

「XX番、のテーマで書いて」と言われたら以下を全て実行する。

### 1. 記事ファイル作成
- `blog/XX.html` を作成
- テンプレートは `blog/06.html` を参考にする（同じHTML構造CSSヘッダーフッター固定バー）
- GA4: G-XJ4YX48ZBW を必ず設置
- LINE CTA: https://lin.ee/quzUszF を必ず設置
- OGP meta タグを設置（og:title, og:description, og:url, og:image）
- XLINEシェアボタンを記事末尾に設置
- 関連記事リンク（related-links）を設置
- カラー: ピンクベージュゴールド系（CSS変数は既存記事と同じ）
- フォント: Cormorant Garamond + Zen Kaku Gothic New

### 2. blog/index.html にカードを追加
- 最後のカードの `</a>` の直後に新しいカードを追加
- カードはインラインスタイルで記述（既存カードと同じ形式）
- タグタイトル概要文日付を含める

### 3. 前の記事のナビリンクを更新
- 1つ前の番号の記事（例: 07.html）を開く
- `nav-links` の `<div>` に「次：新記事タイトル 」のリンクを追加

### 4. git commit & push
```
git add -A
git commit -m "blog: XX番 記事タイトル"
git push
```

## SEO地域キーワード戦略

### 基本方針
- 「地域名＋デリヘル＋求人」のビッグワードはポータルが独占。正面突破は不可能。
- 勝てるのは「地域名  悩みニーズ」のロングテール。ポータルが作らない記事。
- 「蒲田で働ける」という切り口は自店にしか書けない。

### 3層キーワード
- 広域（東京/神奈川）: 検索多いが競合強い。悩み系を付けて差別化。 07, 08でカバー済み
- 中域（横浜/川崎/大田区）: 地域特化で勝機あり。 07でカバー済み
- 詳細（蒲田/大森/品川/大井町）: 競合少なくCV高い。 09, 10で狙う

- 各記事のtitle tagとh1に「地域名＋デリヘル＋求人」を含める
